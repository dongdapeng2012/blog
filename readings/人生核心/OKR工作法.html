周会与四象限表
  • OKR
      Object 目标
    • 目标的时间跨度应为季度及以上

    • 目标不应该量化

    • 设定需要努力达成的目标， 不应太难或太易

    • 5/10 自信度， 并定期追踪， 衡量团队认为离目标更近还是更远， 便于保持方向

    • 如果目标太多出现选择困难， 问问自己如果换个CEO他会做什么？

      Key Result
    • 一个O应该有2-3个KR

    • KR需要能量化

  • 状态指标
    • 保持方向

    • 关于O的才看， 与O无关的不做

    • 随时提醒当前目标

  • 本周
      分级P1/P2
    • 每个级别1-3个， 总数不要超过5个

    • 只做与O相关的

  • 未来4周
      项目列表
    • 列出未来四周的项目计划列表

    • 按优先度排列

    其他
  • 庆功会的作用

  • 不知疲倦地提醒当前目标
