Financial Statement Analysis Framework

--------------------

  1. Define the purpose and context of the analysis

  2. collect data

  3. process data

    • DuPont analysis

      • 杜邦分析将ROE分成盈利能力(净利率), 营运能力(总资产周转天数), 财务杠杆(权益乘数)

    • 资本结构

    • 资本配置

      • 分别计算各业务线的投资比例 CapEx% / Assets%
        其中 CapEx% = CapExi / Total CapEx, Assets% = Asseti / Total Asset

      • 比较各业务线的盈利, 对比来看投资比例是否有效

    • 收入质量

      • 现金比例越高, 收入质量越高

      • Balance Sheet accruals ratio for time t = (NOAt-NOAt-1)/[(NOAt+NOAt-1)/2]
        其中NOA=净营运资产

      • CF accruals ratio for time t = (NIt-CFOt-CFIt)/[(NOAt+NOAt-1)/2]

    • CF relationships

    • 公司价值的分解 decomposition

  4. analyze / interpret data

  5. conclusions and recommendations

    • support for an investment 吸引投资的优势

    • causes for concern 需要关注的劣势

  6. follow up
