投资决策条件

--------------------

  • NPV

    • NPV = 每年 CF 的按规定利率 r 折现的现值 之和

    • 如果 NPV>0, 可以投资

  • IRR

    • IRR = 令 NPV=0 的 折现率

    • 如果IRR大于现行利率则可以投资

  • Payback Period 回收期

    • 不关心折现问题

    • 回收完成的那一年按照比例计算时间

  • 折现回收期 discounted payback period

    • 关心每年的CF折现

    • 回收完成的那一年按照比例计算时间

  • 平均会计收益率 Average Accounting Rate Of Return

    • ARR = Average NI / Average BV

  • 盈利指数 Profitability Index

    • PI = 第二期之后的各期CF的现值 / 初始投资 = 1+(NPV/初始投资) = (NPV + 初始投资) / 初始投资

CF 预测

--------------------

  • 扩张项目

    • 初始支出 init outlay

      • Init Outlay = FCInv + NWCInv

      • FCInv 固定资产投入, investment in new fixed capital
        NWCInv 营运资本投入, investment in net working capital

    • 经营阶段 - 每年的税后CFO annual after-tax CFO

      • CFO = EBIT×(1-T) + D = (S-C-D)×(1-T) + D = (S-C)×(1-T) + T×D

      • S, 销售收入 sales
        C, 现金经营费用 cash operating expenses
        D, 折旧费用 depreciation charge
        T, 税率

    • 结束阶段 - 终期税后非经营现金流 Terminal year after-tax non-operating CF

      • TNOCF = 出售资产的收入 + 初始营运投入 - 出售得利的税费
        = SalT + NWCInv -T(SalT - BT)

      • SalT, 终期固定资产的残值 cash proceeds (salvage value) from sale of fixed capital on termination date
        BT, 终期固定资产的账面价值 book value of fixed capital on termination date

  • 替代项目

    • 初始支出 init outlay

      • Init Outlay = FCInv + NWCInv - Sal0 + T(Sal0 - B0)

      • 替代项目的初期投入额外多出一个处置旧有资产的输入CF = Sal0 - T(Sal0 - B0)

    • 经营阶段 - 增量CFO

      • 增量CFO = ΔEBIT × (1-T) + ΔD = (ΔS-ΔC-ΔD)×(1-T) + ΔD = (ΔS-ΔC)×(1-T) + T×ΔD

    • 结束阶段 - 终期税后非经营现金流 Terminal year after-tax non-operating CF

      • TNOCF = 出售资产的收入 + 初始营运投入 - 出售得利的税费
        = ΔSalT + NWCInv - T(ΔSalT - ΔBT)

      • 初期曾将原项目的资产处置掉, 所以末期将使用处置资产的变动

    • 总体而言相当于在替代项目的各期减掉各期的原项目的CF

  • 通胀的影响-通胀高于预期

    • 盈利能力低于预期

    • 基于低通胀预期的折旧金额小, 税费增加, 实际NPV小于预期

    • 公司债券实际支出贬值, 对公司有利

    • 通胀同时增加收入和成本, 但幅度和公司议价能力有关, 议价能力越高, 成本增加的越少, 销售收入增加的越多

不同存续期的互斥项目

--------------------

  • 最小公倍数法 least common multiple of lives approach

    • 项目A有2年, 项目B有3年, 假设连续投入A项目(相同初始金额)三次共6年, 或连续投入项目B两次(相同初始金额)共6年

  • 等效年金法 equivalent annual annuity approach, EAA

    • 将NPV转化成等效年金

风险分析

--------------------

  • 独立分析 stand along methods

    • 敏感性分析 sensitivity analysis

      • 仅有一个参数变化

    • 情景分析 scenario analysis

      • 设置最好/一般/最坏的情景, 调整多个参数, 分别分析结果

    • 模拟法 simulation / Monte Carlo

      • 将参数认为是分布, 研究结果的分布

  • 市场风险分析 market risk methods

    • 资本预算中的NPV公式的折现率是风险调整后的折现率, 不是公司WACC

    • RProject=Rf+βProject[E(Rmarket)-Rf]

    • 如果项目风险比公司风险高, 如果用WACC代替项目风险调整利率, 会高估项目净现值

实物期权 Real Options

--------------------

  • 分类

    • 延迟投资期权 timing options, 允许公司推迟投资

    • 项目规模期权 sizing options, 允许根据项目规模决定是否扩张或放弃

      • abandonment option, 在项目期间可以按约定价格处置项目

      • growth options, 在期间持续扩张

    • 灵活期权 flexibility options, 除了放弃和扩张之外的选择权

      • price-setting options, 在一定条件下可以改变价格

      • production-flexibility, 公司可以控制产量, 生产其他产品等

    • 基本面期权 fundamental options, 项目收益取决于基本面的期权

      • oil等, 很多R&D项目

  • 实物期权的四个常用方法

    • 将无RO和有RO的项目的NPV对比, 选择较高的

    • NPV = 无RO的NPV + ( RO盈利 - RO成本 )

    • 决策树

    • option pricing model

常见资本预算错误 pitfall

--------------------

  • 没有考虑经济效应 economic responses

  • 用错资本预算模板 template errors

  • 领导偏爱的项目 pet projects

  • 过于关注短期投资决策, 意图影响EPS, NI, ROE

  • 在NPV和IRR冲突时, 应该选用NPV

  • CF估计错误

  • 管理费 overhead costs 估计不准

  • 折旧率应该用项目的风险调整后的利率, 不要用公司WACC

  • 预算策略, overspending / underspending 资本预算

  • 没有考虑其他投资决策

  • 错误的理解机会成本和沉没成本

其他收入计量和估值模型

--------------------

  • Economic Income and Accounting Income

    • Economic Income = 税后CF + ΔMV, 考虑了MV变化, 克服通胀影响

    • 经济折旧, 基于MV变化而减少的部分

    • 经济融资成本, 在CF中忽略, 体现在折旧率中

    • Account Income= 收入-费用 = EBIT × (1-T) + D

    • 会计折旧, 基于历史成本计提的固定资产减值

    • 会计融资成本, 减去之后得到净利润

  • Economic profit, EP

    • 也叫经济增加值 EVA economic value added

    • EP = 税后净营业利润 NOPAT - $WACC = EBIT × (1-T) - WACC × 总资本

    • MV Added, MVA = NPV = Σ[EPt / (1+WACCt))]

    • 投资价值 V = MVA + 投资资本

  • Residual Income, RI

    • RI = NI - Equity charge = NI - 初始投资(start BV of equity) × 要求回报率

    • NPV = Σ[RIt / (1+re))]

    • 投资价值 V = NPV + 原始投资 BVE

  • 声称定价法 claims valuation

    • 计算公司价值, 而非项目价值
